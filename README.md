# D2-event-listener

D2 Event Listener

- Add your .env

```
WALLET_PRIVATE_KEY="a24...ab63"
SLACK_WEBHOOK_URL="https://hooks.slack.com/services/.../.../AbA" #optional
```

- Run (development)

```
npm run dev
```

- Run (production)

```
npm run build && npm start
```
